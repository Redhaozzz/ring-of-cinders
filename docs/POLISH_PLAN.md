# Ring of Cinders 游戏体验打磨计划

*从游戏设计师视角出发的完整优化方案*

---

## 一、第一印象分析（First-Time User Experience）

### 当前问题

1. **主菜单过于简陋**
   - 纯黑背景，缺乏氛围感
   - 没有游戏主题视觉传达
   - 缺少音乐/环境音

2. **零引导**
   - 玩家不知道游戏目标
   - 不知道如何操作
   - 核心机制（闭环焚炉）需要自己摸索

3. **视觉反馈不足**
   - 地面是纯色，缺乏战场感
   - 蚁穴可能还是占位符
   - 砖块库存没有UI显示

4. **音效单薄**
   - 只有基础的攻击/受伤音效
   - 缺少环境音和BGM
   - 焚炉模式缺少戏剧性音效

---

## 二、优化目标

**核心体验定位：** 
> "简单上手，策略深度" — 让玩家在30秒内理解玩法，在3分钟内体验到闭环焚炉的爽快感

**量化目标：**
- 新玩家3次尝试内能完成首个闭环
- 首局游戏时长2-5分钟
- 胜利时有明确的成就感

---

## 三、打磨任务分解

### Phase 1: 视觉完整性（P0 - 必须有）

#### 1.1 地面瓦片系统
**任务：** 替换纯色背景，增加战场氛围
- [ ] 生成地面瓦片素材（草地/土地基础）
- [ ] 实现瓦片平铺渲染
- [ ] 添加边缘装饰（石头、杂草等）

**验收标准：** 地面有纹理感，不再是纯色

#### 1.2 蚁穴美术
**任务：** 为蚁穴创建专属美术
- [ ] 生成蚁穴素材（土堆/洞穴风格）
- [ ] 蚁穴受伤状态变化
- [ ] 蚁穴被摧毁动画

**验收标准：** 蚁穴看起来像蚁穴，不是几何图形

#### 1.3 砖块库存UI
**任务：** 显示玩家可用砖块数量
- [ ] UI位置设计（建议右下角）
- [ ] 显示当前砖块数/最大值
- [ ] 冷却时间指示

**验收标准：** 玩家随时知道还能放几块砖

---

### Phase 2: 新手引导（P0 - 必须有）

#### 2.1 操作提示
**任务：** 首次进入游戏时显示操作说明
- [ ] 半透明覆盖层显示按键
- [ ] WASD移动 / 空格攻击 / K放砖
- [ ] 3秒后自动消失或按任意键跳过

**验收标准：** 新玩家不用猜如何操作

#### 2.2 目标提示
**任务：** 告诉玩家游戏目标
- [ ] 开场文字："消灭所有蚁穴"
- [ ] 第一次放砖时提示："用砖块围住蚁穴"
- [ ] 首次形成闭环时高亮庆祝

**验收标准：** 玩家理解核心机制

---

### Phase 3: 游戏感觉打磨（P1 - 大幅提升体验）

#### 3.1 焚炉模式强化
**任务：** 让闭环焚炉更有戏剧性
- [ ] 火焰粒子效果增强（更多、更亮）
- [ ] 边界砖块发光效果
- [ ] 闭环形成时的"点燃"音效
- [ ] 持续燃烧的环境音
- [ ] 屏幕微微泛红/热浪效果（可选）

**验收标准：** 玩家形成闭环时有"wow"的感觉

#### 3.2 攻击反馈强化
**任务：** 让攻击更有打击感
- [ ] 击中敌人时屏幕微震（2-3px, 50ms）
- [ ] 击杀敌人时小爆炸/烟雾效果
- [ ] 音效随连击变化音调

**验收标准：** 攻击不再感觉"软绵绵"

#### 3.3 音效完善
**任务：** 补充缺失音效
- [ ] 放置砖块音效
- [ ] 砖块冷却完成提示音
- [ ] 蚁穴被摧毁音效
- [ ] 蚂蚁出生音效（可选）
- [ ] 背景音乐（低沉、紧张）

**验收标准：** 游戏有声音层次感

---

### Phase 4: 菜单与流程（P1）

#### 4.1 主菜单美化
**任务：** 营造游戏氛围
- [ ] 背景图（火焰/灰烬主题）
- [ ] 标题动画效果
- [ ] 背景音乐

**验收标准：** 主菜单传达游戏氛围

#### 4.2 游戏结束优化
**任务：** 增强胜利/失败感
- [ ] 胜利时烟花/火焰庆祝效果
- [ ] 显示游戏统计（用时、击杀数、砖块使用数）
- [ ] 失败时画面变灰/慢动作

**验收标准：** 结局有仪式感

---

### Phase 5: 进阶体验（P2 - 锦上添花）

#### 5.1 难度平衡
- [ ] 调整蚂蚁生成速度
- [ ] 调整蚁穴血量
- [ ] 调整玩家血量/伤害

#### 5.2 关卡系统（可选）
- [ ] 多关卡设计
- [ ] 逐步增加蚁穴数量
- [ ] 关卡选择界面

---

## 四、执行顺序

```
Week 1:
├── Day 1-2: Phase 1.1 地面瓦片 + 1.2 蚁穴美术
├── Day 3: Phase 1.3 砖块库存UI
└── Day 4-5: Phase 2 新手引导

Week 2:
├── Day 1-2: Phase 3.1 焚炉模式强化
├── Day 3: Phase 3.2 攻击反馈
├── Day 4: Phase 3.3 音效完善
└── Day 5: Phase 4 菜单优化

Week 3 (可选):
└── Phase 5 进阶体验
```

---

## 五、验收检查清单

每个Phase完成后的自检：

### Phase 1 完成后
- [ ] 截图对比：地面不再是纯色
- [ ] 蚁穴有专属外观
- [ ] 砖块数量可见

### Phase 2 完成后
- [ ] 新玩家测试：是否能在无指导下理解操作
- [ ] 是否能在3次内完成首个闭环

### Phase 3 完成后
- [ ] 视频录制：闭环形成是否有戏剧性
- [ ] 音频检查：声音层次是否丰富

### Phase 4 完成后
- [ ] 完整游戏流程：菜单→游戏→结束→重玩

## 六、风险与备选

| 风险 | 影响 | 备选方案 |
|-----|-----|---------|
| 美术素材生成失败 | 视觉不完整 | 使用程序化图形+颜色变化 |
| 音效生成困难 | 音频单薄 | 使用更多程序化音效 |
| 性能问题 | 粒子效果卡顿 | 减少粒子数量 |
